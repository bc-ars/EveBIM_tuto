## 1. Ouvrir une maquette numérique au format IFC

Commençons par ouvrir la visionneuse. Nous utilisons dans ce tutoriel la visionneuse EveBIM.

Les fonctionnalités expliquées dans ce tutoriel se retrouvent dans la plupart des visionneuses existantes.

Pour ouvrir EveBIM à partir du bureau, vous double-cliquez sur l’icône « EveBIM ». Le logiciel s’ouvre.

A l’ouverture d’EveBIM vous êtes susceptible de trouver le module “modèle” ouvert, nous nous y intéresserons dans

le tuto “xxx”.

En attendant, “fermez” ce module :

![image5]https://github.com/bc-ars/plateform-images/blob/main/evebim/image5-0245376.jpg)

Nous allons maintenant ouvrir la maquette numérique architecturale au format IFC qui a été fournie dans le dossier de consultation.

Si vous avez téléchargé le dossier de consultation sur votre poste de travail, vous pouvez cliquer sur l’onglet “Accueil” puis sur “Ouvrir”. Vous sélectionnez alors le fichier IFC de la maquette architecturale dans votre arborescence.![image6]https://github.com/bc-ars/plateform-images/blob/main/evebim/image6.jpg)

## 2. S’approprier la géométrie extérieure du projet

Nous allons commencer par voir les fonctionnalités de base qui permettent de comprendre la géométrie globale du projet.

Dans la fenêtre principale vous voyez la maquette numérique que vous avez ouverte.

Cliquez dans cette fenêtre en maintenant le “clic gauche” de votre souris et bougez votre souris dans la direction vers laquelle vous voulez faire tourner la maquette. Vous pouvez faire tourner la maquette sur elle-même en répétant l’opération plusieurs fois. Vous pouvez également la faire tourner de haut en bas.

![image-20210505224922826]https://github.com/bc-ars/plateform-images/blob/main/evebim/image10.jpg)

Pour faire “glisser” la maquette, vous maintenez le clic de la molette enfoncé et vous déplacez votre souris dans la direction souhaitée.

![image-20210505225359660]https://github.com/bc-ars/plateform-images/blob/main/evebim/image11.jpg)

Pour zoomer et dézoomer, vous utilisez la molette de votre souris, avant.... Arrière....

*Si vous zoomez plusieurs fois vous pouvez rentrer dans le bâtiment en traversant les murs*

![image-20210505225409244]https://github.com/bc-ars/plateform-images/blob/main/evebim/image12.jpg)

Si vous voulez repositionner la maquette dans sa position initiale, vous pouvez cliquer sur le bouton « voir modèle courant » qui se trouve dans les outils « vue 3D » dans le bandeau supérieur.

![image13]https://github.com/bc-ars/plateform-images/blob/main/evebim/image13.png)

## 6. Prendre des côtes

La vue de l’étage nous permet de prendre des côtes à l’intérieur du bâtiment.

Zoomons sur la maquette à l’aide de la molette de la souris.

Les volumes bleus que vous voyez sont les espaces, vous pouvez les masquer en faisant clic droit sur la souris et en

sélectionnant « cacher » puis « cacher tous les espaces ».

Prenons la cote d’un mur intérieur : on sélectionne tout d’abord le bouton « mesures » dans les outils « vue 3D ».

## 7. Faire une coupe dans le bâtiment

SI vous avez besoin de faire une coupe à un endroit particulier du bâtiment, il est possible de le faire à l’aide de l’outil
« plan de coupe » qui se trouve au niveau des outils « vue 3D » dans le bandeau supérieur.
Cliquez sur la flèche à droite de « plan de coupe », vous pouvez alors choisir votre plan de coupe : XY (plan horizontal
le long de l’axe Z) , XZ (vertical le long de l’axe Y)) ou YZ (plan vertical le long de l’axe X).

Choisissons le plan de coupe XY :


```
Tutoriel 1 - Je m’approprie le dossier
```
Le plan de coupe apparaît, il est représenté par un cercle vert.

Pour déplacer le plan de coupe, il faut maintenir le clic gauche de la souris enfoncé pointé sur le disque vert. Quand il
devient rouge, vous faites glisser la souris dans la direction voulue, de bas en haut dans le cas présent.


```
Tutoriel 1 - Je m’approprie le dossier
```
Pour revenir à la situation initiale cliquez sur “Plan de coupe”


```
Tutoriel 1 - Je m’approprie le dossier
```
Recliquons sur “Plan de coupe” et choisissons cette fois ci le plan XZ.

Le plan de coupe apparaît, représenté par le cercle vert.


```
Tutoriel 1 - Je m’approprie le dossier
```
Comme précédemment, pour déplacer le plan de coupe, il faut maintenir le clic gauche de la souris enfoncé pointé

sur le disque vert, et quand il devient rouge, vous faites glisser la souris dans la direction voulue, en avant ou en

arrière dans le cas présent.



Vous pouvez alors zoomer sur la zone spécifique que vous souhaitez regarder à l’aide de la molette de votre souris. Il

## 9. Capturer des images

Vous pouvez avoir besoin de capturer des images de la maquette pour illustrer votre mémoire technique dans le but
de valoriser votre travail ou pour mettre en évidence un point spécifique.

Pour faire cela, vous cliquez sur le bouton “capture d’écran” dans les outils “vue 3D”.


```
Tutoriel 1 - Je m’approprie le dossier
```
Une fenêtre s’ouvre avec la capture de la fenêtre principale. Vous avez le choix de coller l’image dans le presse-
papier ou de l’enregistrer au format .png.

Une fois dans votre document Word, vous faites CTRL + V pour coller l’image si vous l’avez sauvegardée dans le

presse papier ou vous l’insérer à l’aide de la fonction “insérer / image / cet appareil” et vous allez la chercher dans
l’arborescence à l’endroit où vous l’avez sauvegardé.


```
Tutoriel 1 - Je m’approprie le dossier
```
